# hermes-content-recycle ♻️

將現有 YouTube Shorts 自動跨平台發布到 Twitter/X、Facebook、Instagram、Threads等，
最大化現有內容的 ROI。

## 變現方式

| 方法 | 收益 | 時程 |
|------|------|------|
| 多平台流量 → 聯盟行銷 | 間接 $ | 立即 |
| 更多曝光 → YouTube 訂閱成長 | 長期收益 | 持續 |
| 每個平台都能放聯盟連結 | 直接 $ | 需各平台帳號 |

## 使用方法

Workflow 每日自動掃描 pixabay-shorts-bot 產出的最新 Shorts，
產生跨平台發布文案。

## 平台文案格式

- **Twitter/X**: 短連結 + hashtags
- **Facebook**: 完整敘述 + 引用
- **Instagram**: Bio 連結引導
- **Threads**: 精簡版
- **Telegram/Discord**: 自動通知

## 收入追蹤

所有發布記錄寫入 `data/content_log.json`，
可被 hermes-make-money 收入儀表板讀取。
